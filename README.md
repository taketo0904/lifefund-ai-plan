# 建築AI経営研究会 アップセル戦略ダッシュボード

株式会社LIFEFUND が運営する **建築AI経営研究会**（月額40,000円・税別）の
アップセル商材戦略・カスタマージャーニーを可視化・編集するダッシュボード。

> 目的：建築×AIのリーディングカンパニーとしてのポジション確立
> 〜 月例研究会 → 個別コンサル → 事業変革 → M&A仲介 まで一貫した縦の伴走モデル設計

## 主要ページ

| ページ | 内容 |
|---|---|
| `/` | KPI・3つの差別化レバー・商材サマリー |
| `/economics` | **商材ラインナップ**（Tier 0〜4 + スポット・全項目編集可） |
| `/journey` | **カスタマージャーニー**（認知 → Tier 0 → 1 → 2 → 3 → 4 の遷移条件・編集可） |
| `/kpi` | KPIツリー（仮データ） |
| `/roadmap` | ロードマップ（仮データ） |
| `/team` | チーム体制（仮データ） |
| `/risks` | リスク管理（仮データ） |
| `/hypotheses` | 仮説検証（仮データ） |
| `/strategy` | 戦略ドキュメント（仮データ） |
| `/settings` | テーマカラー・ナビ並び替え |

編集はすべてブラウザの localStorage に自動保存されます。
